# Boutique Artisanale en Ligne

Ce projet est un site web pour une boutique artisanale proposant des créations faites main.

## Structure du Projet

```
.
├── assets/
│   ├── css/
│   │   └── style.css
│   ├── js/
│   │   └── main.js
│   └── images/
├── index.html
└── README.md
```

## Technologies Utilisées

- HTML5
- CSS3
- JavaScript
- Font Awesome (pour les icônes)
- Google Fonts

## Installation

1. Clonez ce dépôt
2. Ouvrez `index.html` dans votre navigateur

## Fonctionnalités

- Page d'accueil avec présentation des produits phares
- Galerie de produits par catégories
- Système de panier
- Page À propos
- Formulaire de contact
- Blog/Actualités
- Design responsive
